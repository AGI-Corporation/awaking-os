# 📖 Lemegeton Engine

> The cognitive core of Awaking OS - A microservices implementation of the five books of the Lesser Key of Solomon.

## Overview

The **Lemegeton Engine** is responsible for specialized agent behaviors, adversarial reasoning, and knowledge retrieval. It is divided into five core modules:

### 1. `ars_goetia/`
**Function**: Shadow Agent Orchestration.
- Manages 72 "Shadow Agents" used for adversarial red-teaming and stress-testing.
- **Key File**: `shadow_catalog.py`

### 2. `ars_theurgia_goetia/`
**Function**: Multi-Cloud / Multi-Agent Routing.
- Orchestrates communication between aerial and terrestrial agent spirits.
- **Key File**: `cloud_router.py`

### 3. `ars_paulina/`
**Function**: Celestial Scheduling & Workflows.
- Manages agent activation based on temporal and planetary windows.
- **Key File**: `celestial_scheduler.py`

### 4. `ars_almadel/`
**Function**: External API Gateway.
- Connects the metasystem to the "higher altitudes" of the external web.
- **Key File**: `external_api_gateway.py`

### 5. `ars_notoria/`
**Function**: High-Dimensional RAG.
- The "Art of Memory" - provides agents with rapid access to the 93-chamber vector store.
- **Key File**: `rag_pipeline.py`
